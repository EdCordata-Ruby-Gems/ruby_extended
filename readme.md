## Documentation
Documentation [available here](/documentation/readme.md).

## Installation
Add this line to your application's Gemfile:
```ruby
gem 'ruby_extended'
```
And then execute:
```bash
bundle
```
Or install it yourself as:
```bash
gem install ruby_extended
```
